# 마찰로 인해 원자 하나가 떨어져 나올 수 있을까?

## 1. 탐구 동기
금속 물체의 겉의 원자는 조금이라도 영향이 있으면 떨어질 가능성이 있다고 생각함

## 2. 관련 과학 개념
원자 사이의 결합을 끊기 위해선 결합 에너지보다 더 큰 에너지가 필요하다.
예를들어 철에는 1g에 10^22개의 원자가 있다.그러므로 개별 질량은 매우 작다.

## 3. 과학적 분석
손톱이나 이로 금속을 긁을때 생기는 힘은 결합 에너지를 넘지 못하므로 집단적으로 떨어저 나갈 가능성은 낮다.
그래도 힘을 집중시켜주면 극히 일부의 원자는 덜어져 나갈 가능성이 없다고 볼순 없다.

## 4. 결론
이론적으론 가능하나 현실적으로 관측 가능한 수준이 아니다.
원자 몇개 정도는 영향을 거의 미치지 않기 때문이다.
이 탐구를 통해 과학은 볼수 없는 차이를 논리로 설명할수도 있는 학문임을 알았다.
